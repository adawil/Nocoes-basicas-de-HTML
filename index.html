Tags de Título

As tags <h1> até <h6> são tags utilizadas para indicar títulos, sendo a tag h1 a de maior relevancia (usualmente chamada de título) e a tag h6 a de menor relevancia. Auxiliam na navegabilidde, bem como no ranqueamento dos sites nos mecanismos de busca


Tags de Texto

Possuem a função de organizar, destacar e marcar alguns pontos do texto, facilitando sua leitura. As tags mais usadas são:

  • <p>

    • Usada para indicar um parágrafo de texto

  • <mark>

    • Usada para destacar um aspecto do texto, de forma análoga a um marca texto manual

  • <small>

    • Usada para deixar o texto menor

  • <i>

    • Usada para deixar o texto em itálico

  • <u>

    • Usada para deixar o texto em sublinhado

  • <u>

    • Usada para deixar o texto em negrito

  • <sub>

    • Usada para deixar o texto subscrito

  • <sup>

    • Usada para deixar o texto sobrescrito

  • <del>

    • Usada para deixar o texto tachado


Listas

As tags de listas servem para listar itens. Elas são representadas por <ol> quando se tratam de listas ordenadas ou <ul> quando se tratam de listas não ordenadas Os itens da lista são identificados pela tag <li>.

Exemplo de lista ordenada

  1. Item A
  2. Item B
  3. Item C
  4. ...

Exemplo de lista não-ordenada

  • Item A
  • Item B
  • Item C
  • ...

Links

Os links servem para conectar páginas ou partes de um mesmo site ou conectar com um site externo. Os mesmos podem ser feitos por meio da tag <a> indicando qual o destino do link, por exemplo "<a href="endereçodedestino.com.br"> palavra clicável </a>"".


Linhas

Muitas vezes pode ser útil insierir uma linha página visando melhorar a visualçização da mesma. O comando para inserir uma linha é <hr>
